# gd Documentation

`gd` is a Rust command line client for GitCode API v5. The documentation is
organized as a compact book set so users, maintainers, and release operators can
find the right level of detail quickly.

## Books

- [Book 1: User Guide](01-user-guide/README.md): installation, authentication,
  command workflows, and scripting.
- [Book 2: Architecture](02-architecture/README.md): CLI layers, API client,
  credential handling, and command boundaries.
- [Book 3: Operations](03-operations/README.md): CI, release, repository
  settings, and quality gates.
